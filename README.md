# NetStream 🎬

A full-stack movie streaming application.

## 🔥 Features
- Add & view movies
- Watch trailers
- Search & filter

## 🛠 Tech Stack
- Spring Boot (Backend)
- MongoDB (Database)
- HTML, CSS, JavaScript (Frontend)

## 📁 Structure
backend/ → API  
frontend/ → UI
